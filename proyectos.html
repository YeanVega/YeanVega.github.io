2025–2026 JavaScript Chart.js Climatología

Monitor de Índices Climáticos y Temperatura Superficial del Océano

Dashboard interactivo para seguimiento ENSO · ONI · RONI · SST

Herramienta de visualización interactiva para el seguimiento en tiempo real de los principales índices climáticos: ONI (Oceanic Niño Index), RONI (Relative ONI) y la temperatura superficial del océano (SST) con sus anomalías. Permite identificar fases ENSO (El Niño, La Niña, Neutro), analizar series históricas y explorar patrones de variabilidad climática que condicionan los pronósticos hidroclimáticos. Construida en JavaScript puro con Chart.js para máxima portabilidad y sin dependencias de servidor.

Series históricas ONI y RONI con bandas de fase ENSO Temperatura superficial del océano y anomalías Identificación automática de fases (El Niño / La Niña / Neutro) Gráficos interactivos con zoom y tooltips Sin dependencias de servidor — HTML autocontenido

02
2025–2026 Python Automatización HTML

Task Monitor — Seguimiento de tareas automatizadas

Dashboard de monitoreo para pipelines Python en producción

Herramienta de monitoreo en tiempo real diseñada para supervisar el estado de tareas programadas y pipelines de procesamiento automatizados en entornos Python/Windows. Muestra el estado de ejecución de cada tarea (completada, con error, detenida o pendiente), el progreso de sub-procesos, los logs de ejecución y los tracebacks de errores — todo desde un único HTML generado automáticamente al final de cada ciclo de procesamiento.

Marcador de estado por tarea (ok / error / detenida / pendiente) Progreso de sub-tareas secuenciales Log de ejecución expandible por tarea Traceback de errores con resaltado Scoreboard global de ejecución Generación automática desde Python
